# ExNulla Stream Ops

Operational repo for the ExNulla dev-ops / job-search livestream.

## What lives here
- **Tools**: OBS browser overlays and helper scripts (`tools/`)
- **Exports**: OBS scene/profile exports and Streamer.bot exports (`exports/`)
- **Docs**: blueprint, runbooks, platform setup notes (`docs/`)

## Quick start
### OBS overlay: Starting Soon countdown
File: `tools/overlays/countdowns/starting-soon-5m.html`

Recommended OBS Browser Source settings:
- Local file
- ✅ Refresh browser when scene becomes active
- ✅ Shutdown source when not visible

## Repo intent
This repository is designed to be reproducible, portable across machines, and auditable (changes tracked via commits).
