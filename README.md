# ExNulla Stream Ops

Operational repo for the ExNulla build-in-public dev-ops / job-search livestream.

## Socials

Canonical list: `docs/socials.md`

## What lives here

- **Tools**: OBS browser overlays and helper scripts (`tools/`)
- **Exports**: OBS scene/profile exports and Streamer.bot exports (`exports/`)
- **Docs**: checklists, platform notes, runbooks (`docs/`)
- **Scripts**: repeatable automation helpers (`scripts/`)
